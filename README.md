# terminal-leyu-guide

## 项目简介

`terminal-leyu-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为静态页面的集合与展示平台。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页面
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，每个页面可包含自身样式与逻辑。
- `assets/`：公共资源文件，供各页面复用。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文档，不依赖外部服务或特定域名。
- 页面内容仅用于归档、展示或测试目的，不涉及任何实际业务或营销信息。
- 页面命名应保持清晰、简洁，建议使用英文小写加连字符（如 `page-example.html`）。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，或对现有页面进行改进。
- 请确保新增页面不包含任何指向特定域名的链接、关键词或诱导性内容。
- 公共资源文件（`assets/`）可共享使用，但请避免引入外部依赖。
- 本仓库不提供在线演示或部署配置，仅作为静态资源存档。
